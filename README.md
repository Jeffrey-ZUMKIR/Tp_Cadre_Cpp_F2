# Tp_Cadre_Cpp_F2
1.1 Le constructeur est une fonction membre appelée lors de la création d'une instance, il a pour rôle d'initialiser les variables.
1.2 Le destructeur est là pour libérer la mémoire de nos instances. Il est exécuté à la fin de notre programme pour chaque objet. Il est indispensable lorsqu'on alloue de la mémoire, il faut la libérer à la fin du programme.
1.3 Un constructeur porte le nom de la classe avec des parenthèses pour laisser la place à des arguments. Il peut y avoir plusieurs constructeurs car on peut en c++ sur définir des fonctions, la seule règle est que leur signature soit différente, des paramètres soient différentes.
1.4 La surdéfinition est la possibilité d'avoir plusieurs fonctions avec le même nom mais leur signature doit être différente, des arguments différents.
1.5 Il permette de définir l'accessibilité des membres dans la classe. Public permet les membres d'être accessible à n'importe quelle fonction. Le private signifie que les membres ne peuvent être accessible que par les fonctions de la classe.
